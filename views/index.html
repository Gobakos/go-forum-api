API Documentation

GET /

Entry point for the application

GET /v1/api/users/

Fetches a list of users (Need authorization header with JWT)

(Authorization: Bearer JWT)

GET /v1/api/users/:id

Fetches a user by ID (Need authorization header with JWT)

(Authorization: Bearer JWT)

DELETE /v1/api/users/:id

Deletes a user by ID (Need authorization header with JWT)

(Authorization: Bearer JWT)

POST /v1/api/auth/login

Logs in a user and returns a JWT

POST /v1/api/auth/register

Registers a new user
